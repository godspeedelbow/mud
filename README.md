MUD reminiscing with node & redux

Getting started
---------------

- `npm i`
- `npm run start`
- `telnet localhost 1337`

Test
----

- `npm run test`

Dependencies
------------

- the latest version of [telnet](https://www.npmjs.com/package/telnet) is unfortunately not on NPM, so using git url as dependency instead.
